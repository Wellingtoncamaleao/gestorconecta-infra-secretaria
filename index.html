Secretaria

Gateway centralizado de mensagens WhatsApp via Evolution API

Base URL

https://0800.gestorconecta.com.br

Autenticacao

Todas as requisicoes exigem header:

X-Api-Key: sua_api_key_aqui

Cada servico/origem tem sua propria API key cadastrada no banco.

Endpoints

POST /api/enviar.php

Envia uma mensagem WhatsApp.

{
  "destino":      "55890000000000",       // obrigatorio — telefone com DDI
  "tipo":         "texto",                  // texto | imagem | documento | audio
  "conteudo":     { "texto": "Mensagem" },  // obrigatorio — varia por tipo
  "prioridade":   "normal",                 // opcional: baixa | normal | alta | urgente
  "agendar_para": "2026-03-02T08:00:00-03:00", // opcional — envia no horario
  "metadata":     { "pedido_id": 123 },      // opcional — dados extras (logado)
  "callback_url": "https://..."              // opcional — recebe POST com status
}

Resposta: 201 (enviado) | 202 (agendado) | 4xx/5xx (erro)

POST /api/enviar-lote.php

Envia varias mensagens com intervalo anti-ban.

{
  "mensagens":    [ ...array de mensagens... ], // max 50
  "intervalo_ms": 3000                          // delay entre envios (ms)
}

GET /api/log.php

Consulta historico de mensagens.

/api/log.php?origem=monitor-projetos&ultimas=24h
/api/log.php?status=erro&de=2026-03-01&ate=2026-03-02

Parametros: origem, destino, status, de, ate, ultimas (24h, 7d, 30d)

GET /api/status.php?id=xxx

Retorna dados completos de uma mensagem pelo ID.

GET /health

Health check. Retorna ok (sem autenticacao).

Exemplo PHP

// Copiar/colar em qualquer projeto PHP
function enviarWhatsApp($texto, $destino, $apiKey) {
    $ch = curl_init('https://0800.gestorconecta.com.br/api/enviar.php');
    curl_setopt_array($ch, [
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_POST => true,
        CURLOPT_POSTFIELDS => json_encode([
            'destino' => $destino,
            'tipo' => 'texto',
            'conteudo' => ['texto' => $texto]
        ]),
        CURLOPT_HTTPHEADER => [
            'Content-Type: application/json',
            'X-Api-Key: ' . $apiKey,
        ],
    ]);
    $resp = curl_exec($ch);
    curl_close($ch);
    return json_decode($resp, true);
}

// Uso:
$resultado = enviarWhatsApp('Teste!', '55890000000000', 'sec_xxx_...');
